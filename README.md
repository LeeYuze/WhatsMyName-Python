# Usage ğŸ“–
Super simple and ready to go. These are the only commands you need:

- **Help command**:
```
python3 whatsmyname.py -h
```

- Check the **number of sites** currently supported in WhatsMyName:
```
python3 whatsmyname.py -c
```

- See the **full sites list** from WhatsMyName:
```
python3 whatsmyname.py -f
```

- Scan the full list for a **single target username**:
```
python3 whatsmyname.py -u anonymouse
```

- **Single site search** for a target username (example: check if secretmonkey is on eBay):
```
python3 whatsmyname.py -u secretmonkey -s ebay
```
